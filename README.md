Pegasus
===================
Abstract
---------
Unicorn-based multiarchitecture disassembler and debugger.

Current Objectives
-------------------
1. Universal Multiplatform GUI (Qt or Electron)
2. Blazing fast
3. Disassembly subsystem using Capstone Project (C++ or Rust?)
4. Extremely modular and customizable
